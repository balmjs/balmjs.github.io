# 额外文件

# extras.includes

extras.includes: string[] = []

打包时需要从根目录下附加额外文件。

# extras.excludes

extras.excludes: string[] = []

打包时需要从根目录下排除额外文件。

🌰 举个栗子：

module.exports = {
  extras: {
    excludes: ['.extra', 'extra.txt']
  }
  // 其他配置项...
};

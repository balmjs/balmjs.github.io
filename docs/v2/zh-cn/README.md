# BalmJS [![NPM version][balm-image]][balm-url]

> 一个灵活的前端工作流方案

## 什么是 BalmJS ？

BalmJS 规范了前端的最佳实践和工具，以帮助开发者保持高效工作。

| [balm-gui](https://github.com/balmjs/balm-gui)        | N/A                                                    | BalmJS 可视化工具                      |

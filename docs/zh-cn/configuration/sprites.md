# Sprite

### `sprites.basePath`

`string`

Path to use in CSS referring to image location. Defaults to `'../'`.

### `sprites.padding`

`number`

Amount of pixels to include between images or svgs. Defaults to `1`.

### `sprites.mode`

Output modes: `css`, `view`, `defs`, `symbol`, `stack`. Defaults to `'css'`.
